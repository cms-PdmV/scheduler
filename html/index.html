======= Scheduler

Scheduler reads "CSV" files.
Working directly with excel go to "file"->"save as"->choose csv format and comma as separator.
Working with csv directly do not leave empty spaces between comma separated values.
First row is column names (author,groups,priority,events,keywords,time, deadline).
Order that column names are listed does not matter.

Required columns and values:

  • author -> name of an author, can be none.
  • groups -> groups that request belongs to, can be none.
  • priority -> has to be between 1 and 10, wrong values default to priority = 1.
  • events -> number of total events in request, has to be above ~30000, otherwise request will not be added.
  • keywords -> variety of values separated by comma (like: status, prepid, tags and etc.). In excel entered to a single field, in csv wrapped with quatation marks.
  • time -> time in seconds it takes to proccess one event in a system.

Optional columns and values:

  • deadline -> date until request has to be completed, format of date "year-month-day hour:minutes:seconds" (for example: 2014-01-01 20:00:00).
